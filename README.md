## NetOpsy

> NetOpsy is part of NetHydra (Formely known as HydraPWK)

is free and Open-source linux distributions (Live) based on [NetHydra](https://nethydra.github.io). NetOpsy is focused for `Digital Forensic Investigations`.
it's come with `CONFIGURATION` and Free software from Various Open-source project.

### Documentations

- [build](https://github.com/NetHydra/NetOpsy/blob/main/docs/build.md) - Build your own NetOpsy image.
- [Installation](https://github.com/NetHydra/NetOpsy/blob/main/docs/install.md) - Installation instructions to install NetOpsy on the device.
